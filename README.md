# 🧮 Scientific Calculator

A fully functional scientific calculator built with HTML, capable of handling advanced mathematical operations.

## 🚀 Features

* Basic arithmetic operations (+, −, ×, ÷)
* Scientific functions (sin, cos, tan, log, sqrt)
* Keyboard input support
* Responsive and clean UI
* Error handling for invalid expressions
* Handles complex expressions with parentheses and operator precedence
* Real-time calculation display

## 🛠️ Tech Stack

* HTML

## 📝 Description

A clean, functional scientific calculator that supports both basic and advanced mathematical operations with a responsive interface.
